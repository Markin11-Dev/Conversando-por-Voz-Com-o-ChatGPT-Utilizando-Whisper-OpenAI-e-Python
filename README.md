# Conversando-por-Voz-Com-o-ChatGPT-Utilizando-Whisper-OpenAI-e-Python
Sistema que combina as tecnologias de Speech-to-Text (voz para texto) e Text-to-Speech (texto para voz) para proporcionar uma conversa multi-idiomas por voz com o ChatGPT, algo possível graças ao Whisper. Para aqueles que ainda não conhecem, o Whisper é uma tecnologia de Reconhecimento Automático de Fala (ASR) da OpenAI
