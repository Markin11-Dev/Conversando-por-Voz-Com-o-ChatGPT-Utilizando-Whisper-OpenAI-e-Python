## 🧠 Sobre o Projeto

O **Voice AI Assistant** é uma aplicação que demonstra um pipeline completo de Inteligência Artificial multimodal:

> 🎙️ Voz → 🧠 Texto → 💬 IA → 🔊 Voz

A solução permite que o usuário converse com um modelo de IA usando apenas a voz, sem necessidade de digitação.

---

## 🚀 Motivação

A maioria das interfaces ainda depende de texto, mas o futuro da interação humano-computador está na voz.

Este projeto foi criado para explorar:

- Interfaces naturais  
- IA generativa aplicada  
- Experiência conversacional

## 🔄 Arquitetura do Projeto


Usuário (voz)
↓
🎤 Captura de áudio (JavaScript)
↓
🧠 Whisper (Speech-to-Text)
↓
💬 GPT (Processamento de linguagem)
↓
🔊 gTTS (Text-to-Speech)
↓
Resposta em áudio
